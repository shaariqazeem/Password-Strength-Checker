# Password Strength Checker (Python)

A Python-based tool that analyzes password strength based on multiple security criteria such as length, uppercase/lowercase letters, numbers, and special characters.

## Features
- Evaluates password strength (Weak, Medium, Strong)
- Uses regex for pattern matching
- Simple and interactive CLI

## Tech Used
- Python
- re (Regular Expressions)

## How to Run
python password_checker.py
